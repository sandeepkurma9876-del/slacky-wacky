# slacky-wacky

A fully functional Slack bot built specifically to run 24/7 on Hack Club Nest (using Socket Mode via `@slack/bolt`).

## Features
- `/slacky-wacky-ask [prompt]` - Ask the Groq AI a question or give it a prompt to generate text.
- `/slacky-wacky-status` - Check if the bot is alive, its uptime, and runtime stats.
- `/slacky-wacky-help` - Show the help message.
