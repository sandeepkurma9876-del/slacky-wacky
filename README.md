# slacky-wacky

A fully functional Slack bot built specifically to run 24/7 on Hack Club Nest (using Socket Mode via `@slack/bolt`).

## Features
- `/ask-groq [prompt]` - Ask the Groq AI a question or give it a prompt to generate text.
- `/bot-status` - Check if the bot is alive, its uptime, and runtime stats.
- `/bot-help` - Show the help message.
